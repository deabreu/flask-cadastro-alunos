# flask-cadastro-alunos
# Aplicação para cadastro de alunos
---
## Requisitos

- Python 3.7
- Virtual Environment
- Pip

## Instalação

```
$ virtualenv venv
$ source venv/bin/activate
$ pip3 intall -r requirements.txt
$ flask run
```

## Acesso

### Tela principal

No navegador acessar

```
http://localhost:5000/
```
